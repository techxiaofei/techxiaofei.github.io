Centos7安装SSR教程

2020年8月6日
| Others
| 阅读

原文地址：https://left-pocket.github.io/

记录一下在centos7下安装ssr(shadowsocksr)的教程。

安装必要依赖包

yum install git -y
yum install epel-release -y
yum install libsodium -y
yum install epel-release -y
yum install libsodium -y

下载shadowsocksr

git clone -b manyuser https://github.com/shadowsocksr-backup/shadowsocksr.git && cd shadowsocksr/

修改配置

vim user-config.json

{
    "server": "0.0.0.0",
    "server_port": 80,
    "local_address": "127.0.0.1",
    "local_port": 1080,

    "password": "wtffff",
    "method": "aes-128-ctr",
    "protocol": "auth_aes128_md5",
    "protocol_param": "",
    "obfs": "tls1.2_ticket_auth_compatible",
    "obfs_param": "",
    "speed_limit_per_con": 0,
    "speed_limit_per_user": 0,

    "additional_ports" : {},
    "additional_ports_only" : false,
    "timeout": 120,
    "udp_timeout": 60,
    "dns_ipv6": false,
    "connect_verbose_info": 0,
    "redirect": "",
    "fast_open": true
}

启动客户端要配置的加密、协议、混淆一定要对的上。

启动服务

cd shadowsocks/ python server.py -d start

这可能是最简单的教程了，简单明了。
