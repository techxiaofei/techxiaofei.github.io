背景

随着ChatGPT的快速走红，科技巨头们也纷纷入局AI之战。目前看来动作最快的要数微软了，不仅曾大手笔投资于OpenAI，更火速推出了新必应 AI。

新必应可以基于搜索生成内容、数据是实时的，而ChatGPT只能数据只训练到2021年，而且并没有联网，是一个离线的AI聊天工具。意味着新必应可以咨询最新的热点，而ChatGPT则只能"瞎编"了。

新必应AI的用户体验更好，会给你提供数据源的地址，同时会给你推荐最有可能问的问题的答案供你选择。

我一番体验下来，这是我的感受，整体来说比ChatGPT体验好不少。

  1. 新必应AI响应非常快，比ChatGPT快不少。
  2. 新必应AI会通过搜索引擎搜索互联网答案，实时通过AI模型生成答案。
  3. 生成答案的同时还会提供答案的源地址，方便追踪。
  4. 会给你推荐几个后续问题，你可以点击按钮直接问，也可以在搜索框了自己问问题。

申请

下面我就来教大家如何申请微软新必应AI，国内无需魔法上网就可以使用微软的网站和服务，所以你也能轻轻松松的使用微软新必应AI。

Edge浏览器

新必应AI需要使用微软的Edge浏览器，使用其他浏览器无法使用。

进入以下链接： https://www.microsoft.com/en-us/edge/download

然后根据你的系统选择下载不同的版本：

访问新必应地址

下载安装之后进入浏览器，输入新必应访问地址： https://www.bing.com/new

如果你的IP在国内或者你的设置里面的国家/地区选的中国，会自动跳转到国内特供版bing搜索，这个时候 新必应AI 是不能用的。

下载ModHeader插件

这个时候我们就需要模拟国外的IP，只需要去Edge扩展商店下载 ModHeader 插件，这个插件的作用就是给你的请求头上加一个IP地址，模拟国外IP访问bing。

插件下载之后打开，在插件里面输入：

名字就是：X-Forwarded-For

IP填写为：4.2.2.2 （微软DNS）

修改之后重新访问 https://www.bing.com/new，如果依旧跳转到之前的页面，需要在必应搜索页面的右上角的设置里面修改国家/地区，随便修改到某个其他地方即可。

加入候补名单

当你做完这两个修改，再次输入以上地址 https://www.bing.com/new

就会跳转到新必应简介的页面，并有一个加入候补名单的按钮，我们点击加入候补名单，如果你已经有微软账号，直接点击登录即可，如果没有，点击注册。推荐使用 微软outlook邮箱注册。

加入候补名单成功后会收到一封邮件（使用注册/登录所用的微软邮箱地址登录邮箱查看）。

你在新Bing的候补名单上! 一旦我们准备好让你尝试，你就会收到一封电子邮件。

成功加入新必应

需要等候几个工作日，我这边只等候了两天就收到新的邮件。

欢迎来到新必应。我们很高兴为您提供新必应的早期预览-您的人工智能答案引擎。

聊天

我们在邮件里面点击 Chat Now或者输入以上新必应AI地址就可以进入新必应页面聊天了。点击立即聊天。

我们来尝试跟新必应AI对话：

  1. 新必应AI响应非常快，比ChatGPT快不少。
  2. 新必应AI会通过搜索引擎搜索互联网答案，实时通过AI模型生成答案。
  3. 生成答案的同时还会提供答案的源地址，方便追踪。
  4. 会给你推荐几个后续问题，你可以点击按钮直接问，也可以在搜索框了自己问问题。
  5. 点击新主题可以停止此次对话，开启新的对话。

由于chatGPT是2021年之前的数据且不连接互联网，所以完全没办法拿到最新热点新闻，在这点上比搜索引擎起家的新必应 AI要逊色不少。

体验

与ChatGPT相比，新版必应的反应更快，回复耗时更短；此外，新版必应还更加“人性化”，不仅会为用户提供问题选项，还会在回复中使用表情符号；同时，新版必应更加“严谨”，会在回复中标注所引用的来源，并列出对应网站。
