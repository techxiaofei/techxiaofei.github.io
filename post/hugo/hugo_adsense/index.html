从零开始搭建个人博客网站系列 六、接入google adsense广告创收

by 科技小飞哥
2022-12-10
约 1596 字 - 预计阅读 4 分钟
hugo

本篇博客的视频教程首发于 Youtube：科技小飞哥。

前言

使用个人博客也有一年多的时间，这一年多的时候踩了很多坑，也学到了很多知识。 这里分享一个系列教程：从零开始搭建个人博客，把我的经验分享在这里。
系列地址：https://techxiaofei.com

背景

网站建好了，也小有流量了，下一步就可以申请Google Adsense了，简单来说就是允许谷歌在你的网站上投放一些广告，根据用户的浏览和点击来向你付费。

还有一些其他的广告联盟，比如百度广告联盟，不过需要进行备案，我的网站不再国内的阿里云/腾讯云等云平台，无法备案。所以选择了Google Adsense，这也是大部分人的选择。

这其中我折腾了半年，提交了十几次，终于申请Google Adsense广告成功，所以以后我也是有收益的人了。

下面分享一下怎么申请Google Adsense来带来收益。

步骤

在Google Adsense关联你的网站

第一步就是登录Google Adsense平台来关联你的网站，官网地址是：https://www.google.com/adsense/start/ ，点击右上角的 Sign up 用你的谷歌账号登录并绑定Adsense账号就可以了，

注册需要填写你的网站地址、邮箱、以及一些选项。

关联网站的方式其实就是在每个html页面的header里面都注入同一行 javascript 脚本，这个脚本用来标明你的身份，这样谷歌才知道是谁。

由于我使用的是hugo静态博客模版，直接在模版的header里面就可以了。

我使用的是even主题， 在/themes/even/layouts/particals/head.html 里面添加就行。

其他的主题应该也是类似的目录。放在这里的话，每次执行hugo命令生成的所有目标html文件，的<head></head>里面都会把这段script添加进去，这就是模版的作用。

最简单的方式就是在上述位置任意一个位置添加Google Adsense要求你添加的代码。

1
2

<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-xxxxxxxxxxxxxxxx"
     crossorigin="anonymous"></script>

其中 client=ca-pub-xxxxxxxxxxxxxxxx 是你的client id，唯一标识你的身份的。

如果你想简单点，以上的改动就足够了，编译上传到Github，你的每一个页面都会有上述的Script代码。

当然为了方便config配置，我支持了config配置

1
2
3
4

{{ if .Site.Params.googleAd }}
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client={{ .Site.Params.googleAd }}"
     crossorigin="anonymous"></script>
{{ end }}

然后在config.toml里面配置

1
2
3

[params]
//...其他的一些配置
googleAd = "ca-pub-xxxxxxxxxxxxxxxx"      # Google Ad

非常方便。

发布代码到网站之后点击申请审核，如果google检测到你的网站有这个代码，就可以开始审核了，一般审核一周左右会告诉你结果，可能失败也可能成功。

审核通过

说实话，我审核了半年，十几次被拒绝。都心灰意冷了。

像上图这样，他不会给你具体的原因，只会说需要注意哪些方面才能申请通过。

我个人的感觉就是：保持原创，保持质量，保持更新。一二十篇就可能通过了。

我一开始的博客都是知乎，个人博客，微信公众号一起发。估计谷歌检测出来这些内容网上已经有了，觉得不是你的原创，通过的概率就小一些，所以没给我审核通过，后面的十来篇博客我都没有发知乎，只发了个人网站。然后边发边申请。

然后再试了几次，失败了之后，最后死马当活马医的心态又申请了一次，终于通过了。

收益

申请成功通过之后，可以看到主页变成了这样。

我的网站没什么流量，所以收益很少，流量多的话，收益很多。看起来是广告展示和点击都有收益，不过展示的收益非常低，有人点击你的收益才会高起来。

配置广告模式

点击广告-概览我们可以配置广告模式，默认的按网站，点击修改。就可以修改在哪里展示广告。

可以设置

  • 页内广告
  • 锚定广告
  • 穿插广告
  • 侧边栏广告

还可以打开自动广告，Google 将自动在您网站上的所有最佳位置展示广告。

剩下的就不停的更新你的网站，增加流量，等收益吧。

来点一下我的个人博客网站给我增加下收益吧。 https://www.techxiaofei.com/
