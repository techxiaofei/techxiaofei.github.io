背景

我知道很小伙伴想使用ChatGPT，但是经常被IP限制，今天就给大家分享一个无需登录，完全免费的GPT4的网站。

同时如果你想要自己搭建一个这样的网站，只需要三分钟就可自己搭建一个不用魔法，不用登录的基于GPT4的AI网站。

打开这个界面，是不是非常熟悉，这不就是基于GPT4的微软New Bing AI聊天机器人吗？

而这实际上是我自己基于new bing的API和Cookie搭建的GPT4网站，内容是原滋原味的New Bing AI，甚至还内嵌了很多Prompt提示词。可以说非常强大。

免费使用

如果你想体验直接点击 我的New Bing AI网站 即可体验。

如果上述链接无法访问，可以使用原作者提供的链接 访问。

如果上述链接依然无法使用，可以点击 Github源码 找到最新链接访问。

搭建

如果你也想搭建一个这样的AI聊天机器人，非常简单，只需要不到3分钟就可以一键部署。而且无需云服务器，无需魔法。

Github账号

我们首先在 Github 上访问这个地址：go-proxy-bingai

然后点击右上角的Fork也就是克隆，然后点击Create Fork把这个仓库克隆到你的账号下

这个时候就会把这个github 仓库克隆到你的个人账号下面。

Vercel账号

克隆Github仓库之后我们进入Vercel平台。Vercel 直接使用刚才的Github账号登录并授权。

我们直接点击Add New->Project，然后找到我们的go-proxy-bingai这个Github仓库，点击Import导入

这个project name我们可以随便选，使用默认的即可。

然后直接点击Deploy等待一段时间就可以完成部署。

添加域名

部署完成之后我们就可以直接使用了，默认是使用的Vercel给我们提供的域名，但是这个域名有时候在国内访问受限，我们需要配置自己的域名。

我们在当前项目里面点击Setting设置选项。

然后点击左边的菜单栏Domain添加域名，这里可以添加顶级域名也可以添加二级域名，我这里添加的是二级域名 bing.techxiaofei.com。

添加完会提示非法配置，我们需要在域名解析商里面添加记录。

添加一条A记录，然后名称是你的二级域名的值，我使用bing前缀，IP地址就是Vercel平台提供的的IP地址76.76.21.21。

添加之后就可以使用你的二级域名 bing.xxx.com 访问了。

添加Cookie

这个网站原作者默认提供了一些Cookie，如果出现：**很抱歉，你似乎已注销。**说明Cookie已过期，我们需要重新设置Cookie，点击右上方的设置图标，然后点击设置用户，这个输入框填写的就是Cookie的值。

这个Cookie怎么获得呢？非常简单。

打开 http://www.bing.com 网站，按F12进入开发者模式，点击应用程序，左边的Cookie->www.bing.com，然后找到 _U的值，就是Cookie，复制即可。

然后回到刚才的设置用户界面，输入刚才的Cookie保存，然后点击下方的新主题打开一个新的主题即可重新聊天。

结尾

现在你就可以无限制使用自己搭建的ChatGPT网站了。赶紧尝试下吧。
